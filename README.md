# FaustAI — русскоязычный AI-ассистент с LLM на вашем устройстве

**FaustAI** — это форк популярного приложения PocketPal AI, адаптированный под русскоязычных пользователей и оптимизированный для работы на Android-устройствах.  
Всё, что вы пишете, остаётся на вашем устройстве. Никаких облачных серверов, никакой передачи данных — полная приватность и свобода.

## 🌟 Возможности

- **Полная офлайн-работа** — загрузите GGUF-модель и общайтесь без интернета.
- **Русский интерфейс** — всё переведено, включая системные сообщения и настройки.
- **Поддержка популярных моделей** — Qwen, Llama, Mistral, Gemma и другие в формате GGUF.
- **Стриминг ответов** — токены появляются по мере генерации (как в ChatGPT).
- **Настройки генерации** — температура, top_p, контекст, количество потоков.
- **Управление моделями** — скачивание, удаление, импорт из файловой системы.
- **Режим разработчика** — возможность сохранять сгенерированный код и выполнять его через Termux (в разработке).
- **Полный контроль** — вы владеете кодом и можете модифицировать приложение под свои нужды.

## 🚀 Установка

### Готовый APK (рекомендуемый способ)

1. Перейдите в раздел **Releases** этого репозитория.
2. Скачайте последний файл `FaustAI-debug.apk`.
3. Установите приложение на ваш Android-планшет или телефон (Android 10+).
4. Откройте приложение, выберите язык **Русский** в настройках.
5. Скачайте модель через встроенный менеджер (или импортируйте свою `.gguf`).

### Сборка из исходников

Если вы хотите собрать APK самостоятельно (например, для внесения изменений), используйте GitHub Actions:

1. Сделайте форк этого репозитория.
2. Перейдите во вкладку **Actions**.
3. Выберите workflow **Build Android APK**.
4. Нажмите **Run workflow** → **Run workflow**.
5. Через 5–10 минут скачайте APK из раздела **Artifacts**.

Либо соберите локально (требуется Android SDK и Node.js 18+):
